CodeStack Conference Live

https://csconf.org/

CodeStack Conference Live (https://csconf.org/) is a website owned and operated by CodeStack Academy. I was given this website to work on during my internship for CodeStack in May of 2021. In collaboration with the lead UI/UX designer on the project, I was given prototypes built in Adobe XD and I was required to build the homepage to the exact specifications from each prototype. The website is built in Angular which reguired skills such as understanding Angular as a framework and having the ability to write efficient code in TypeScript.
